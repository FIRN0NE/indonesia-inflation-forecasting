# Project: Inflation Forecasting BPS

Workspace eksperimen/sandbox milik Rama untuk memproyeksikan inflasi BPS menggunakan Python.

## Struktur Project

- `data/`: Dataset inflasi bulanan BPS M-to-M dari tahun 2010 - 2025.
- `src/`: Modul/skrip Python untuk pengolahan data dan modeling.
- `notebooks/`: Jupyter Notebooks untuk EDA dan visualisasi model.
- `.venv/`: Virtual environment Python.
- `requirements.txt`: Daftar pustaka Python yang digunakan.

## Tools Utama

- Python (dengan pandas, statsmodels, prophet, matplotlib, scikit-learn).
- **Graphify** (`C:\Users\Rama\Documents\project\graphify`) untuk memetakan struktur kode/grafik jika diperlukan.

## Preferensi & Efisiensi Antigravity

- **Bahasa**: Indonesia.
- **Gaya Komunikasi**: Singkat, to-the-point, kurangi narasi berlebihan (over-explanation).
- **Token Saving**: Merujuk langsung ke file atau script yang ada tanpa menulis ulang kode besar jika tidak diminta.
- Simpan cache atau data olahan di folder `data/` demi konsistensi.
